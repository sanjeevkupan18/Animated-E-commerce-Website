# SOLESTEP — Awwwards-Inspired Shoe E-Commerce

A visually stunning frontend-only shoe brand website built with React (Vite), Tailwind CSS, and GSAP.

## Quick Start

npm install
npm run dev

Open http://localhost:5173

## Build for Production

npm run build
npm run preview

## Features

- GSAP Animations: ScrollTrigger, page transitions, text reveals, parallax
- Custom Cursor: Neon dot + lagging ring with hover effects
- Animated Navigation: GSAP clip-path overlay with staggered menu items
- Page Transitions: Accent-color curtain wipe on every route change
- CLI Chatbot: Terminal-style floating chatbot (/help, /products, /price, /about)
- Cart System: Add/remove/update quantities, total calculation
- SEO: Meta tags, semantic HTML, robots.txt, sitemap.xml

## Design System

- Accent Color: #C8FF00 (electric lime)
- Display Font: Bebas Neue
- Body Font: DM Sans

## Tech Stack

- React 19 + Vite 8
- Tailwind CSS v4
- GSAP + ScrollTrigger
- React Router DOM v7
- Context API
